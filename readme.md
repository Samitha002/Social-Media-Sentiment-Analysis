

**social Media Sentiment Analysis Project**



**Project Overview**



This project focuses on analyzing social media tweets to understand public sentiment (Positive, Negative, Neutral). I built an End-to-End Data Pipeline using SQL, Python, and Power BI to transform raw data into actionable business insights.

 **Key Features**



**Data Management:** Successfully loaded and managed 500+ social media tweets in a MySQL database.



**Sentiment Processing:** Used Python to fetch data from SQL and categorize sentiments.



**Interactive Visualization:** Developed a Power BI Dashboard featuring:



**Map Visuals:** To track sentiment by geographic location.



**Donut Charts:** For an overall percentage split of sentiments.



**Slicers:** To allow users to filter data dynamically.



**Geographic Insights:** Identified specific regions with high negative or positive feedback.
